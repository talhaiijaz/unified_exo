# EEG Lab Guide

**Path:** `eeg_lab/`

Neural signal acquisition tools (Pi + Teensy → browser plots).

## Run locally

```bash
cd eeg_lab
./run.sh
```

See `eeg_lab/README.md` for hardware setup.

## Integrating with unified server

EEG lab can run standalone or be linked from the main website via new pages in `nanotech_website/app/portal/`. Open a PR to add portal navigation when ready.
